Game Development

Blue Screen


Blue Screen is a stealth / action shooter, created in Unity in under 48 hours. All graphics, music, sfx, and code were made from scratch during the time limit. The gameplay consists of the user navigating across two worlds. In the real world, the player must sneak around to avoid drones. In cyberspace, the game plays like a descent-style shooter.

Audhumbla


Audhumbla is a co-op bullet-hell shooter. The game supports a variety of inputs, and does not have a player cap. This video is of a five-player run of the first level. Created in Love2D.

Crossing Beams



Crossing Beams is a multiplayer mobile game, created for the Cardinal Game Jam. The game requires three players, who combine beams to match the color of enemies. Created in Unity.

God Complex


A god game, developed for the 2014 Global Game Jam. The goal is to keep your civilization alive, but you can only perform actions that harm them. Gameplay consists of throwing minor disasters at the villagers, thus convincing them to prepare for later, uncontrollable disasters. Created in Unity.

The Damned


Local multiplayer game created in Unity. Players compete to stay alive from falling rocks. However, everybody is chained together.

Space Board Game

Prototype of a browser-based multiplayer, turn-based strategy game. Can be player either asynchronously or in real-time. Uses Three.js and Socket.io.

Submersion


An multiplayer, virtual-reality game, set in dueling submarines. Created for CS 248 in Unreal Engine 4.

Graphics Programming

Non-Euclidean GPU Raytracer


A Non-Euclidean raytracer, developed in C++ and OpenGL. Lua scripting was embedded in order to animate game objects. A level editor was created in Python.

The Sixth Sun


As part of an unfinished game, I implemented a hex-tiled planet construction. It could be passed textures for different tile types, and outputs a graph for pathfinding, etc.

Math4D Visualization Software


Basic visualization software for multidimensional functions, made in about 2 days. Created using pyOpenGL.

Terrain Implementations


As part of a science fair project, I implemented and compared several terrain rendering algorithms. This included an Geometry Clipmaps, implemented in the Ogre3D engine.

Voxel Rendering


Simple Voxel render with basic optimizations. Created in C++ and OpenGL.
